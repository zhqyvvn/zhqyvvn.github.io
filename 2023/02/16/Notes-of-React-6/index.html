Study Notes of React (6) - this.props

Posted on 2023-02-16 Edited on 2023-03-14 In Study Notes , React Views: Word count in article: 865

A component can pass information to another component. Information that gets passed from one component to another is known as props.

Every component has props. A component’s props is an object. It holds the information that passed to that component.

4. Pass an Event Handler as a prop

Functions can be passed as props. It is common to pass event handler functions.

Event handler can be defined as a method in the component class, just like the render method, without the function keyword.

prop attributes will work with names that follow the JavaScript variable name rules.

The naming convention for a event handler is adding handle before the event type, like handleClick, and for a prop, it is adding on before the event type, like onClick.

Names like onClick only create event listeners if they’re used on HTML-like JSX elements. Otherwise, they’re just ordinary prop names.

5. this.props.children

Every component’s props object has a property named children. this.props.children will return everything in between a component’s opening and closing JSX tags.

Components can be self-closing tags, such as <MyComponentClass />, and they can also be <MyComponentClass></MyComponentClass>.

this.props.children would return everything in between <MyComponentClass> and </MyComponentClass>.

In Example 1, <BigButton>‘s this.props.children would equal the text, “A child of BigButton.”
In Example 2, <BigButton>‘s this.props.children would equal a <LilButton /> component.
In Example 3, <BigButton>‘s this.props.children would equal undefined.

If a component has more than one child between its JSX tags, then this.props.children will return those children in an array. However, if a component has only one child, then this.props.children will return the single child, not wrapped in an array.

{this.props.children} between <ul></ul> tags inside the return function of the List component makes the list-items display.
