1. Stateless Functional Components

In React, components can also be defined as JavaScript functions — Function Components, which is different from Class Components.

The function component should return the same JSX that was originally returned by the render() method. render(){} is removed in the function component.

2. Function Components and Props

To access props, function component needs a parameter named props.

Within the function body, access properties of props using props.propertyName, without this keyword.
