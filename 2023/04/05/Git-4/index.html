9. Prune Stale Branches

There are three level of branches:

  1. Branches on the remote repository. (git push -d to delete one)
  2. Local snapshots of the remote branches. (also called remote-tracking branches)
  3. Local branches, tracking the remote branches. (git branch -d to delete one)

The tracking branch will be automatically deleted when its remote branch was deleted in the same local repository, but if the remote repo was deleted in a different repo, then we need to delete stale remote-tracking branches.
