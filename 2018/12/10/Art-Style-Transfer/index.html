🌟 Built with

• Python
• Flask
• Amazon Web Services
• Tensorflow

🌟 Components

• Style Transfer —> Tensorflow (Fast Neural Style)
• Database —> Dynamodb
• Image Storage —> S3

🌟 Features

• Authentication (login / signup / logout)
• Chose one of the art styles and perform style transfer to users’ uploaded photos
• Browse all saved processed photos and their full-resolution versions in the album
• Delete one selected photo per time, or delete all photos one time in album

📌 Note

• Fast Neural Style models are in reference to here
• All images used for style transfer examples were randomly selected from Google Image
